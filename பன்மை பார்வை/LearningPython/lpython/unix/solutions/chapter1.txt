1. Interaction. ***************************************************************

% python
copyright information lines...
>>> "Hello World!"
'Hello World!'

2. Programs. ******************************************************************

% cat module1.py
print 'Hello module world!'

% python module1.py
Hello module world!


3. Modules. *******************************************************************

% python
>>> import module1
Hello module world!
>>>


4. Scripts. *******************************************************************

% cat module1.py
#!/usr/local/bin/python        (or #!/usr/bin/env python)
print 'Hello module world!'

% chmod +x module1.py
% module1.py
Hello module world!
